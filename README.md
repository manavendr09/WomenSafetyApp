## 📖 About

**Women Safety App** is a web-based personal safety tool designed to protect women in emergency situations. Unlike traditional navigation apps that find the *shortest* route, this app finds the **safest route** — prioritizing well-lit, populated, and secure paths. In case of danger, a single tap on the **SOS button** instantly shares the user's live location with trusted family members.

## ✨ Key Features

### 🗺️ Safest Route Navigation
- Suggests the **safest path** to the destination, not just the shortest
- Avoids isolated, low-traffic, or poorly lit areas
- Powered by map and geolocation APIs

### 🆘 SOS Emergency Button
- One-tap **SOS button** for instant distress signaling
- Automatically captures the user's **real-time GPS location**
- Sends an **emergency alert** with location link to pre-saved family members
- Fast, simple, and accessible — works even under stress

### 📍 Live Location Sharing
- Shares the user's **live location** with trusted family contacts
- Family members receive an instant notification with a map link
- Ensures help can be dispatched to the exact location

### 👨‍👩‍👧 Family Contact Management
- Add and manage trusted family members or emergency contacts
- Contacts receive alerts automatically when SOS is triggered

## 📱 How It Works

```
1. User opens the app and enters destination
        ↓
2. App calculates the SAFEST route (not shortest)
        ↓
3. User travels with real-time navigation
        ↓
4. If danger arises → User presses SOS button
        ↓
5. App captures live GPS location
        ↓
6. Emergency alert + location link sent to family members
        ↓
7. Family can track and dispatch help immediately
```

## 🌐 Deployment

This project is deployed on **Netlify**. To deploy your own copy:

1. Push your project to a GitHub repository
2. Go to [netlify.com](https://netlify.com) and log in
3. Click **"Add new site" → "Import from Git"**
4. Select your repository
5. Leave build settings blank (plain HTML/CSS/JS)
6. Click **Deploy** 🚀
